# 🏏 Cricket Big Bash League Analysis (2019–2020)

## 📌 Project Overview

This project analyzes the batting performance of players in the 2019–2020 Big Bash League (BBL) season using Microsoft Excel. The objective is to identify performance trends, evaluate player contributions, and develop an interactive dashboard to support data-driven insights into team and individual batting performance.

## 🎯 Objective

To analyze batting performance in the 2019–2020 Big Bash League (BBL) season and identify patterns, trends, and key contributors that influence team success. The project evaluates player consistency, strike rate, boundary-hitting ability, and overall batting impact using Excel dashboards and visualizations.
