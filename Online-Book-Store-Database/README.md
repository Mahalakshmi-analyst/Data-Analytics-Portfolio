# 📚 Online Book Store Database

Course: Managing and Querying Database

Database: online_book_store

## 📌 Project Overview

This project is a complete MySQL-based Online Book Store Database System designed to manage customers, books, orders, payments, reviews, coupons, carts, and wishlists. The project demonstrates database design, SQL querying, automation, and reporting capabilities.
