# 📊 Sales & Performance Analysis

**Course:** Exploratory Data Analysis using Python

---

# 📌 Project Overview

This project analyzes sales performance data to understand business trends across products, customers, regions, and sales channels. 
Using Python for Exploratory Data Analysis (EDA), the project identifies key drivers of revenue and profitability while providing business recommendations to improve pricing strategies,
marketing effectiveness, inventory management, and customer engagement.

---

# 🎯 Business Problem

The business aims to understand overall sales performance across products, customers, regions, and sales channels to identify the factors driving revenue and profit.

The analysis focuses on identifying strengths, weaknesses, and opportunities that support data-driven business decisions.

---

# 🎯 Project Objectives

- Analyze sales trends across multiple years.
- Identify the best-performing products and brands.
- Analyze customer segments and loyalty impact.
- Evaluate the performance of different sales channels.
- Identify top-performing and low-performing products.
- Generate actionable business recommendations to improve sales and profitability.

# 🧹 Data Cleaning & Preprocessing

The following preprocessing steps were performed:

- Removed duplicate records
- Handled missing values
- Standardized date formats
- Standardized category names
- Standardized region names
- Standardized customer information
- Created new features:
  - Profit
  - Profit Margin (%)
  - Discount (%)
  - Year
  - Month
- Corrected text inconsistencies
- Removed outliers in Sales and Profit
- Prepared a clean dataset for analysis

# 💡 Business Recommendations

- Increase inventory for high-performing products.
- Optimize pricing strategies for low-margin products.
- Improve marketing campaigns for underperforming regions.
- Strengthen customer loyalty programs.
- Reduce excessive discounting where it negatively impacts profit.
- Focus promotional campaigns on high-performing sales channels.
