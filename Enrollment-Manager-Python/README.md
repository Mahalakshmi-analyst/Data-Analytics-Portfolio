# 🎓 Enrollment Manager System

**Course:** Programming Using Python

---

# 📌 Project Overview

The Enrollment Manager System is a Python-based application developed to automate and simplify the student admission and enrollment process. 
The system efficiently manages student records, course information, fee details, and enrollment status using Object-Oriented Programming (OOP). 
It minimizes manual work, improves data accuracy, and provides intelligent insights through AI integration.

---

# 🎯 Project Objectives

- Design and develop a Python-based system for managing student enrollments efficiently.
- Automate student registration and course enrollment processes.
- Maintain centralized records of students, courses, and enrollment status.
- Reduce manual effort and minimize data entry errors.
- Provide easy search, update, and deletion of enrollment records.
- Generate reports such as enrolled students per course.
- Ensure data consistency through proper validation during enrollment.
- Improve administrative productivity through automation.

# 🗄️ System Design

The Enrollment Manager System uses a Python-based in-memory data structure to efficiently manage student admissions.

The system organizes data using Python classes and objects, simulating real-world database tables.

It maintains:

- Student admission records
- Course details
- Teacher information
- Fee status
- Enrollment history

Each student record contains:

- Roll Number
- Student Name
- Age
- Mobile Number
- Email Address

Enrollment information includes:

- Course Name
- Assigned Teacher
- Total Fees
- Fees Paid
- Remaining Balance
- Course Status

Admission status is automatically maintained as:

- Enquiry
- Pending
- Enrolled

---

# 📊 Functionalities

- Automatically generates enrollment dates to simulate real admission timelines.
- Supports complete CRUD operations (Create, Read, Update, Delete) for enrollment records.
- Provides separate views to analyze:
  - Confirmed Admissions
  - Pending Fee Students
  - Enquiry Students
- Generates enrollment reports for administrative analysis.
- Designed with a scalable architecture that supports future integration with SQLite or other databases.
- Integrates AI using the Groq API to generate intelligent insights and perform enrollment data analysis.
